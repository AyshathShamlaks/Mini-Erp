# 🛒 Mini ERP System

A full-stack Mini ERP (Enterprise Resource Planning) System developed for small retail shops to simplify product, customer, inventory, and sales management. The application provides an easy-to-use interface for managing daily business operations, generating GST invoices, and monitoring stock levels.

---

## 📌 Project Overview

The Mini ERP System is designed to reduce manual work and improve the efficiency of retail shop management. It allows shop owners to manage products, customers, sales, and inventory from a single dashboard while automatically generating GST invoices.

---

## ✨ Features

### 📦 Product Management
- Add new products
- Update product details
- Delete products
- Manage stock quantity
- Low stock alerts

### 👥 Customer Management
- Add customers
- Edit customer details
- Delete customers
- Store customer information

### 💰 Sales Management
- Create sales records
- Automatic GST calculation
- Invoice generation
- Automatic total price calculation

### 📊 Dashboard

- Sales Analytics

- Search functionality
